# Desafio QA - Beedoo 2026

Este repositório contém a execução do desafio técnico para a vaga de **Analista de Qualidade de Software (QA) Júnior**.

O objetivo do desafio foi analisar e testar o módulo de **cadastro e listagem de cursos** da aplicação disponibilizada.

---

## Estrutura do projeto

- `docs/analise-da-aplicacao.md`  
  Contém a análise da aplicação, fluxos identificados e pontos críticos para teste.

- `casos-de-teste/planilha-de-testes.md`  
  Contém o link para a planilha com cenários, casos de teste e execução dos testes.

- `relatorio-de-bugs/bugs-encontrados.md`  
  Contém o registro detalhado dos bugs encontrados durante os testes.

- `evidencias/`  
  Pasta contendo prints utilizados como evidência da execução dos testes.
